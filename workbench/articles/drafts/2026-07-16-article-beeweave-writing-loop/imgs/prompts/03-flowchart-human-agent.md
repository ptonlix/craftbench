BeeWeave 人与 Agent 的写作分工图

Layout: 16:9 左右双通道。左侧是 Agent 自动化流水线，右侧是作者判断链，中间汇合成一个完整作品包。中央保留清晰负空间，避免信息拥挤。

STEPS LEFT - Agent 承担流程:
- 抓取素材
- 整理文件
- 生成初稿
- 配图与检查
- 记录 trace

STEPS RIGHT - 人承担判断:
- 选择素材
- 确定观点
- 手动改稿
- 授权费用
- 确认规则

MERGE:
- 两条通道在中间下方汇合成四个成果图标：文章、图片、trace、写作规则
- 中心主句只显示：Agent 承担流程，人承担判断
- 用一道明显的人工确认闸门连接两侧，表现 human-in-the-loop

LABELS:
- 只显示上述短标签，以及：BeeWeave 协作写作
- 中文必须准确，不增加解释段落或乱码

COLORS:
- 暖米色背景 #F5E6D0
- 暖橙色 #E8751A 表现 Agent 流程
- 金黄色 #F4A623 表现共同成果
- 深棕色 #4A2C20 表现作者、文字和结构
- 猩红色 #C0392B 表现确认闸门和权限边界
- 色值仅用于渲染，不显示颜色名或十六进制值

STYLE:
- Screen print / silkscreen vintage workflow poster
- 人物使用简化剪影，不画详细面孔
- 平涂色块、粗箭头、半色调网点、纸张颗粒和轻微套色偏移
- 最多五色，无渐变，无 3D，无写实电脑界面
- 中文使用粗窄海报字，核心句最大，其余为短标签
- Clean composition with generous white space
