## 基本信息

- **仓库名**: Easy-Vibe
- **Star 数**: 16K
- **出品方**: DataWhale（国内知名开源学习社区）
- **定位**: Vibe Coding 入门实战

## 学习路线

分三段：

1. **入门阶段** — 做 AI 小游戏，门槛低，先建立手感
2. **进阶阶段** — 产品创意落地和 Web 前端原型
3. **高阶阶段** — 桌面应用和多 Agent 协同工程

## 特点

- 完全用实战项目驱动
- 从头到尾教「把想法变成产品」
- 对零基础用户非常友好
- 中文友好

## 在学习路线中的位置

作为第一步使用，先跑通第一个项目，建立对 Vibe Coding 的完整认知。之后进入 [[entities/vibe-coding-cn]] 学工作流方法论。
