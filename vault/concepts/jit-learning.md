# JIT Learning

JIT Learning 即 just-in-time learning，强调在新问题出现时快速理解本质、识别关键、要求最高效方法、把控结果质量并预防未知风险。它不要求人长期记忆每个具体技术做法，而要求人能在需要时快速学、正确问、有效判断。

## Key Ideas

- AI 时代具体技术做法变化很快，普通人不必长期系统记忆所有 prompt、harness、agentic loop 细节。
- 人类仍需要理解基本原理和最佳实践，否则无法向模型提出高质量要求。
- 模型不知道人的背景、目标、约束、风险偏好和真实场景；这些信息需要人提供。
- 人的价值从“造轮子”转向“判断该用哪个轮子、装到哪里、会不会出事、能否自动化”。
- 解决问题后，应把经验抽象为稳定认知或自动化流程，其余临时细节可以忘掉。

## Relation To Agent Work

JIT Learning 与 Agent Loop 的关系是：人不必亲手完成所有低层操作，但必须知道应该要求系统建立怎样的 loop、怎样的上下文和怎样的护栏。 ^[inferred]
