## 路线概览

三个免费、开源、中文友好的 GitHub 仓库组成 Vibe Coding 最扎实的自学路线。

## 四个阶段

### 第一步：建认知（Easy-Vibe）

用 [[entities/easy-vibe]] 跑通第一个项目，亲身感受一遍从想法到成品的完整过程。

- 入门：AI 小游戏，建立手感
- 进阶：Web 前端原型
- 高阶：桌面应用和多 Agent 协同

### 第二步：学工作流（vibe-coding-cn）

用 [[entities/vibe-coding-cn]] 掌握标准化流程，从此告别随手丢一句需求然后翻车的日子。

- 标准化 Prompt 撰写
- 个人 Skill 库沉淀
- 完整工程 Workflow 搭建

### 第三步：补工具链（awesome-vibe-coding）

用 [[entities/awesome-vibe-coding]] 根据自己场景挑趁手的工具。

### 第四步：用真实项目收尾

从小工具做起，走一遍「想法 → AI 生成 → 运行调试 → 迭代成品」的完整循环，让成果说话。

## 预期时间线 ^[inferred]

- **第一周**：跑通第一个 AI 小游戏，信心大涨
- **第二周**：做出第一个 Web 原型，开始找到感觉
- **第一个月**：掌握标准化工作流，基本不再翻车
- **第二个月**：沉淀出自己的 Skill 库和工作流模板
- **第三个月**：整套 AI 工具链稳定运转，一个人顶一个小团队

## 与付费课的关系

很多 Vibe Coding 付费课程（动辄几千块）的内容 99% 来自这三个免费仓库。课程卖的本质上不是内容，而是信息差的打包服务。^[inferred]
